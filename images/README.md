# images/ — 사진 폴더

이 폴더에 아래 두 개 사진 파일을 넣으세요.

## 필요한 파일

| 파일명        | 용도                          | 권장 크기           |
|--------------|-------------------------------|---------------------|
| profile.jpg  | 히어로 섹션 프로필 사진        | 400×400px 이상 (정방형) |
| avatar.jpg   | AI 채팅 창 상단 아바타 사진    | 100×100px 이상 (정방형) |

## 사용 방법

1. 본인 사진을 이 폴더에 복사합니다.
2. 파일 이름을 위 표에 맞게 변경합니다.
3. VS Code에서 `index.html`을 열고 Live Server로 확인하세요.

## 주의사항

- 사진이 없어도 사이트는 정상 작동합니다 (사진 영역만 숨겨짐).
- jpg 대신 png, webp도 사용 가능합니다. 그 경우 index.html에서
  `images/profile.jpg` → `images/profile.png` 으로 수정하세요.

## GitHub 배포 시

사진 파일을 GitHub 저장소에 함께 push하면 GitHub Pages에서도 표시됩니다.
개인정보가 걱정된다면 .gitignore에 `images/` 를 추가하고 배포 서버에만 업로드하세요.
